# IQ Note Prompt Template

Copy and paste this template, filling in the bracketed `[PLACEHOLDERS]`:

```
Explain [CONCEPT].
File example: [PATH_TO_FILE]
Create a .md file in IQ_Notes with:
- TL;DR summary table
- Full comparison table with clear columns
- Example walkthrough using the file above
- Pipeline/flow diagram
- Breakdown table — what each layer "sees"
- Real-world analogies
Filename: [Concept_Name]_IQ.md
```

## Examples

| You say | I create |
|---------|----------|
| "Explain Compiler vs Interpreter. File example: chapter_01_Basics/01_HelloWorld.js. Filename: Compiler_vs_Interpreter_IQ.md" | ✅ Done above |
| "Explain Synchronous vs Asynchronous. File example: chapter_02_JS_Concepts/async_demo.js. Filename: Sync_vs_Async_IQ.md" | A new note with the same visual style |
| "Explain LocalStorage vs SessionStorage vs Cookies. File example: chapter_02_JS_Concepts/storage_demo.js. Filename: Storage_Comparison_IQ.md" | A new note with the same visual style |

## Tip

You can also say the same thing in natural language — the template is just a shortcut.
